# Demo test

#### Instructions front/client:

###### Prérequis: Node.js, Angular

```bash
cd Project/frontend
```

```bash
npm install
```

```bash
ng build
```

```bash
ng serve
```

###### Aller sur http://localhost:4200

#### Instructions backend/serveur:

###### Prérequis: .NET, C#

```bash
cd  Project/backend
```

```bash
dotnet run
```

---

#### Test :

Ajouter un compte depuis SQL Server ou avec addLogin() une fois dans le `Program.cs`.
